Welcome to the Financial Management System

1. User Roles

This application supports three types of users: Administrator, Manager, and Accountant. Each role has specific permissions and access levels.

2. Login

Users can log in using their credentials. If you forget your password, use the 'Forgot Password' option.

3. Financial Reports

Users can generate financial reports including trial balances, income statements, and balance sheets for specified date ranges.

4. Adjusting Entries

Adjusting journal entries can be created and submitted for approval. Ensure to include proper documentation for each entry.

5. Financial Ratios

The dashboard displays key financial ratios, color-coded to indicate performance levels.

6. Event Logs

All changes made in the system are logged for auditing purposes. You can view these logs under the relevant sections.
